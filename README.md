# 🎯 Event-Driven Programming Final Project

This repository contains our **Final Project for Event-Driven Programming**, developed using **Java**. The project demonstrates the application of event-driven concepts such as user interaction handling, GUI components, and system responsiveness.

📅 **Presentation Date:** April 14, 2026  
👨‍🏫 **Instructor:** Sir Adrian De Jesus  

---

## 📌 Project Overview

The goal of this project is to design and implement a functional Java application that follows the principles of **event-driven programming**. The system responds to user actions (such as button clicks, input fields, and selections) and performs corresponding operations dynamically.

### 🔧 Key Features
- Interactive **Graphical User Interface (GUI)**
- Event handling using **listeners**
- Dynamic data processing based on user input
- Organized code structure for maintainability
- Real-time system response to user actions

---

## 🛠️ Technologies Used

- **Java (JDK)**
- **Java Swing / AWT** (for GUI)
- **Event Listeners** (ActionListener, etc.)
- **Object-Oriented Programming (OOP)** principles
